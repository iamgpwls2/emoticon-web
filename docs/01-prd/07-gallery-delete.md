# 07. Gallery & Delete (PRD)

## 목적
- 사용자의 생성 결과를 갤러리로 보여주고, 삭제할 수 있는 요구사항을 정의합니다.

## 작성 시점
- 2026-06-02

## 작성할 내용(목차)
- 갤러리 목록
  - 정렬/페이징
  - 썸네일/메타데이터(생성 시각, 프롬프트 요약 등)
- 삭제
  - 삭제 확인 UI
  - DB 레코드 삭제 + Storage 오브젝트 삭제 정책
- 권한/보안
  - 타 사용자 리소스 접근/삭제 방지(RLS/Storage policy)
- UX 요구사항
  - 로딩/에러/성공 상태
  - 빈 상태(empty state)
- 수용 기준(AC)

# 05. Image Generation (PRD)

## 목적
- 정제된 프롬프트를 사용해 이미지를 생성하고 결과를 저장/표시하는 요구사항을 정의합니다.

## 작성 시점
- 2026-06-02

## 작성할 내용(목차)
- 생성 트리거/요청 흐름
  - 중복 요청 방지
  - 타임아웃/재시도
- 결과물 규격
  - 해상도/포맷/개수
- UX 요구사항
  - 로딩(진행 상태)
  - 실패(원인/재시도)
  - 성공(미리보기/다음 액션)
- 저장 전략
  - Storage 저장 경로(`user_id/...`)
  - DB 기록(참조/메타데이터)
- 비용/쿼터(선택)
- 수용 기준(AC)

# 04. LLM Prompt Refine (PRD)

## 목적
- 사용자 입력을 기반으로 LLM이 이미지 생성용 프롬프트를 정제하는 단계의 요구사항을 정의합니다.

## 작성 시점
- 2026-06-02

## 작성할 내용(목차)
- 정제 목적/원칙
  - 사용자 의도 보존
  - 금지/위험 콘텐츠 필터링(정책)
- 입력/출력
  - 입력: 사용자 텍스트 + 옵션
  - 출력: 최종 프롬프트 + 메타데이터(선택)
- UX 요구사항
  - 로딩/에러/재시도
  - 결과 프롬프트 확인/수정 가능 여부(선택)
- 보안 요구사항
  - API Key는 backend-only
  - 로그에 민감정보 남기지 않기
- 수용 기준(AC)

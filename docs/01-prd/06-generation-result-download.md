# 06. Generation Result Download (PRD)

## 목적
- 생성 결과를 다운로드할 수 있는 요구사항(권한, UX, 실패 처리)을 정의합니다.

## 작성 시점
- 2026-06-02

## 작성할 내용(목차)
- 다운로드 대상
  - 단일 파일 / 묶음(zip) 여부
- 권한/정책
  - 사용자 본인 리소스만 접근 가능(RLS/Storage 정책)
- UX 요구사항
  - 다운로드 버튼 상태(로딩/비활성)
  - 실패/재시도
  - 성공 피드백
- 구현 고려사항
  - Signed URL / Proxy 다운로드 여부
  - 브라우저 호환성
- 수용 기준(AC)

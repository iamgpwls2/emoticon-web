# 02. Image Upload & Preview (PRD)

## 목적
- 사용자가 이미지를 업로드하고 생성 전 미리보기를 확인하는 기능 요구사항을 정의합니다.

## 작성 시점
- 2026-06-02

## 작성할 내용(목차)
- 사용자 스토리
- 업로드 제약
  - 파일 타입/크기 제한
  - 모바일/데스크톱 고려
- UX 요구사항
  - 드래그앤드롭(선택)
  - 로딩/진행률/취소
  - 실패/재시도/오류 메시지
- 보안/정책
  - Storage 경로 규칙(`user_id/...`)
  - Public/Private 버킷 여부
- 저장/참조 방식
  - 업로드 후 Storage URL/경로를 DB에 기록할지 여부
- 수용 기준(AC)
